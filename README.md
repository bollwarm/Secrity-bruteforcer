## Secrity-bruteforcer

   Secrity::bruteforcer -- supports multi-protocol and services, which is used to 
quick ptentest for weak password of common applications.



## INSTALLATION

To install this module, run the following commands:

	perl Makefile.PL
	make
	make test
	make install
